Studio F

Sound Setup

Room PC

Audio Setup

  1. Turn on the power switch on the audio rack.
  2. Press the "PC" button on the audio control panel.
  3. Adjust the volume using the main volume control.
  4. On the computer, ensure the correct audio output is selected.

Laptop

Laptop Connection

  1. Connect your laptop to the HDMI cable provided.
  2. Press the "Laptop" button on the audio control panel.
  3. Adjust the volume using the main volume control.
  4. Ensure your laptop's audio output is set correctly.

Projector Setup

Room PC Projector

  1. Turn on the projector using the remote control.
  2. Select "PC" as the input source.
  3. The room PC should display automatically.
  4. Adjust the projector focus if needed.

Laptop Projector

  1. Connect your laptop to the HDMI cable.
  2. Turn on the projector using the remote control.
  3. Select "HDMI" as the input source.
  4. Configure your laptop's display settings as needed.
